# Simple Pet API with Go and Postgres

## Project setup

- Clone the project
- Create a `.env` file and fill in the values
- Install postgres as a Docker container with `make start-db`
- Run the project with `make run`

## Project outline

- Pet -> simple CRUD operations for a pet
